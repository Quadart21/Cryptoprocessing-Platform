## [0.14.5] — 2026-06-02

### Добавлено

- **Rate limit API key (на аккаунт)**: общий лимит запросов в минуту для всех ключей tenant (`X-API-Key` на `/api/v1/client/*`) + burst 50/10 с. Настройка: `RATE_LIMIT_API_KEY_ACCOUNT_PER_MINUTE`, `RATE_LIMIT_API_KEY_ACCOUNT_BURST`.
- **Max concurrent connections на IP**: одновременные запросы с одного IP ограничены (по умолчанию 30). Настройка: `MAX_CONCURRENT_CONNECTIONS_PER_IP`, `MAX_CONCURRENT_CONNECTION_TTL_SECONDS`.

## [0.13.2] — 2026-05-31

### Исправлено

- **Бухгалтерия и оборот**: суммы в crypto (DOGE, BTC и др.) пересчитываются в USD/USDT по курсу с наценкой; устранён учёт 1:1 (15 DOGE ≠ 15 USDT).
- **Курсы**: live-запрос CoinLore при отсутствии символа в кэше; ошибка вместо silent fallback при создании инвойса.
- **Crypto-Cash**: глобальная обработка ошибок провайдера (502/400 вместо 500); устойчивый парсинг списка валют; `PAYMENT_PROVIDER=crypto_cash` при деплое, если заданы ключи.

### Изменено

- **Комиссии**: минимум 0,5 USDT (диапазон 10–250 USDT экв.) применяется и к комиссии провайдера (Crypto-Cash), и к наценке платформы; диапазон считается по USDT-эквиваленту после пересчёта из crypto.
- **Оплата**: gross пересчитывается из `amount_crypto` при settlement; статистика админки считает fees от USD-эквивалента инвойса.
